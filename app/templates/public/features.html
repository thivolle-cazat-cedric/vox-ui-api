Notifications

Notifications local

Interne à la page web, lors d'une notification un popup en bas de page s'affichera indiquant la raison de cette aparirion.

Notification de bureau

Pour les navigateurs compatibles au notification de bureau (Chrome, Firefox, Safari), il vous suffi d'avoir une page ouverte (pas forcément active dans votre navigateur), et lors d'une notification elle apparaitra sur votre ordinateur.

Quelles-sont les Notifications

Lors d'une appel entrant

Vous allez connaitre le numéro de l'appelant, et si ce numéro est présent dans votre l'annuaire de l'application, vous connaiterez le nom

Tableau de bord

Le tableau de bord vous permet d'avoir une vue d'ensemble de votre parc téléphonique Voxity. Vous connaiterez donc l'état des postes

Légende

Poste non connecté à la plate-forme voxity
Poste connecté et disponible
Le poste est en train de sonné, ou est en train de lancer un appel
Poste en ligne
État inconnu

Lorsque vous cliquez sur une extension, vous généré un clik2call, si vous cliqué sur la fléche , vous aurez le détail du poste ainsi que ces lignes en cours (dans le cas ou vous etes administrateur)

Raccourcis clavier

Dans toutes les page vous disposez de raccourcie clavier.

r
Sur le dashbord, permet de rafréchire l'état des postes
c
Affiche la fenêtre pour composer un numéro
h
Retour au tableau de bord
esc
Retrir les filtres de recherche lors d'une recherche de contact
